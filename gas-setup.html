신청서가 접수되면 Google Apps Script(GAS)를 통해 자동으로 아래 이메일로 알림이 발송됩니다.

1

Google Apps Script 프로젝트 생성 및 코드 작성

1
Google Apps Script 접속

script.google.com 에 접속합니다. Google(Gmail) 계정으로 로그인이 필요합니다.

2
새 프로젝트 생성

왼쪽 상단 + 새 프로젝트를 클릭하고 프로젝트 이름을 "청솔행정사_이메일알림" 등으로 변경합니다.

3
기존 코드 삭제 후 아래 코드 전체 붙여넣기

편집창의 기본 코드를 모두 지우고 아래 [복사] 버튼으로 복사한 코드를 붙여넣으세요.

Google Apps Script — Code.gs 
// ============================================================
// 청솔행정사사무소 민원 접수 이메일 알림 스크립트
// Google Apps Script (Code.gs)
// ============================================================

/** 알림 수신 이메일 주소 */
const NOTIFY_EMAIL = 'cirrus01@naver.com';

/** 긴급 건 제목 태그 */
const URGENT_LABEL = '🚨 [긴급]';

/**
 * POST 요청 처리 진입점.
 * 신청 페이지에서 fetch(no-cors)로 JSON 데이터를 전송합니다.
 */
function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    if (data.type !== 'new_application') {
      return jsonResponse({ status: 'ignored' });
    }
    sendNotificationEmail(data);
    return jsonResponse({ status: 'ok' });
  } catch (err) {
    Logger.log('Error: ' + err.message);
    return jsonResponse({ status: 'error', message: err.message });
  }
}

/** 이메일 본문 생성 및 발송 */
function sendNotificationEmail(d) {
  const isUrgent  = (d.is_urgent === true || d.is_urgent === 'true');
  const urgentTag = isUrgent ? URGENT_LABEL + ' ' : '';

  const subject = urgentTag + '[청솔행정사] 새 민원 접수 - '
                + (d.category || '미지정') + ' (' + d.app_number + ')';

  const body = '안녕하세요, 정충교 행정사님.\n\n'
    + '새로운 민원 신청이 접수되었습니다.\n\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + ' 접수 정보\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + ' 접수번호   : ' + (d.app_number   || '-') + '\n'
    + ' 접수일시   : ' + (d.submitted_at || '-') + '\n'
    + ' 긴급여부   : ' + (isUrgent ? '⚡ 긴급 처리 요청' : '일반') + '\n\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + ' 신청인 정보\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + ' 서비스 분류 : ' + (d.category      || '-') + '\n'
    + ' 세부 서비스 : ' + (d.sub_service   || '-') + '\n'
    + ' 신청인 유형 : ' + (d.applicant_type|| '-') + '\n'
    + ' 상호/회사명 : ' + (d.company_name  || '-') + '\n'
    + ' 담당자 이름 : ' + (d.manager_name  || '-') + '\n'
    + ' 연락처     : ' + (d.phone         || '-') + '\n'
    + ' 이메일     : ' + (d.email         || '-') + '\n'
    + ' 업종       : ' + (d.business_type || '-') + '\n'
    + ' 처리기한   : ' + (d.deadline      || '-') + '\n'
    + ' 회신방법   : ' + (d.reply_method  || '-') + '\n\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + ' 신청 내용\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n'
    + (d.content || '(내용 없음)') + '\n\n'
    + '━━━━━━━━━━━━━━━━━━━━━━━━━\n\n'
    + '청솔행정사사무소 자동 알림 시스템';

  GmailApp.sendEmail(NOTIFY_EMAIL, subject, body, {
    name: '청솔행정사사무소 알림'
  });

  Logger.log('이메일 발송 완료: ' + subject);
}

/** JSON 응답 헬퍼 */
function jsonResponse(obj) {
  return ContentService
    .createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}

/**
 * ★ 테스트용 함수
 * GAS 편집기에서 이 함수를 선택하고 ▶ 실행하면
 * cirrus01@naver.com 으로 테스트 이메일이 실제 발송됩니다.
 */
function testEmail() {
  sendNotificationEmail({
    type: 'new_application',
    app_number: 'TEST-20250101-0001',
    submitted_at: new Date().toLocaleString('ko-KR'),
    is_urgent: false,
    category: '인허가·등록',
    sub_service: '건축인허가',
    applicant_type: '법인',
    company_name: '(주)테스트기업',
    manager_name: '홍길동',
    phone: '010-0000-0000',
    email: 'test@example.com',
    business_type: 'IT서비스업',
    content: '테스트 신청입니다.',
    deadline: '2025-01-30',
    reply_method: '이메일'
  });
}
위 코드의 NOTIFY_EMAIL 값이 이미 cirrus01@naver.com으로 설정되어 있습니다. 변경이 필요하면 상단 상수만 수정하세요.

배포 > 새 배포

유형: 웹 앱 선택

"유형 선택" 드롭다운에서 웹 앱을 선택합니다.

아래와 같이 설정합니다
항목설정값설명

5
권한 승인

처음 배포 시 Google 권한 팝업이 뜹니다. 계속 → 허용을 클릭합니다.

"Google이 확인하지 않은 앱" 경고가 뜰 수 있습니다. 고급 → (안전하지 않음으로) 이동을 클릭하면 됩니다. 본인이 만든 스크립트이므로 안전합니다.

배포 완료 팝업에서 웹 앱 URL을 복사합니다. 아래 형식입니다:

웹사이트에 GAS URL 등록하기

GAS 웹훅 URL 입력 (빠른 적용)

배포에서 복사한 URL을 아래에 붙여넣고 [저장 및 적용]을 클릭하세요.
이 브라우저 localStorage에 저장되어 신청 페이지에 자동으로 적용됩니다.

영구 적용 권장: apply.html 직접 수정
apply.html에서 const GAS_WEBHOOK_URL = 'YOUR_GAS_WEBHOOK_URL_HERE'; 부분을 찾아 실제 URL로 교체하세요. localStorage 방식은 해당 브라우저에서만 유효합니다.

자주 묻는 문제 & 해결

증상원인 및 해결
이메일이 오지 않음 ① URL이 정확한지 확인
② GAS 배포 시 액세스: 모든 사용자 설정 확인
③ 네이버 스팸함 확인
④ GAS 편집기 실행 로그에서 오류 확인
코드 수정 후 반영 안 됨 코드 수정 후 반드시 새 배포를 다시 진행해야 합니다.
권한 오류 (Permission denied) GAS 편집기에서 testEmail 함수를 직접 실행해 권한을 한 번 승인해 주세요.
발신 한도 초과 무료 Gmail 계정은 하루 100건 한도. 민원 접수량 기준으로 충분합니다.
no-cors 모드 경고 정상 동작입니다. GAS는 CORS 헤더를 반환하지 않아 no-cors 모드를 사용하며, 이메일은 정상 발송됩니다.
